# drx-alert
Kullanım:
/alert maxi 
server_config dosyasından değiştirilebilir.Config.FRL.Commands'a eklemeler yapılabilir.
Çalışması için Steam Hex ID'nizi server_config Config.FRL.Admins kısmından girmeniz gerekiyor.
